Assignment: Baby-ransomware (กลุ่มละ 2 คน)
ให้นักศึกษาเขียนโปรแกรมเข้ารหัสและถอดรหัสไฟล์ตามเงื่อนไขข้างล่าง ด้วยภาษา C/Python/Java/JavaScript เป็นต้น
ข้อกาหนด
• เริ่มต้น นศ. ให้สร้างคู่กุญแจ RSA ที่มี Key ขนาด 2048 bits ไว้ก่อน (จะได้ Private key และ Public key)
• สร้างโฟลเดอร์ประกอบด้วย
➢ โปรแกรม baby-ransomware
➢ ไฟล์รูป (jpg หรือ png)
➢ ไฟล์ text (txt)
• โปรแกรมมี 2 modes
➢ Encryption
➢ Decryption
• 1. เมื่อเลือกโหมด Encryption จะสร้างกุญแจแบบ AES ที่มี Key ขนาด 256 bits (random สร้างใหม่ทุกครั้งเมื่อรัน)
➢ IV (Initialization vector) 16 bytes สาหรับ AES ใช้เป็น Random
➢ แสดงผลกุญแจ AES ที่สร้างออกทางหน้าจอ 1 ครั้ง
➢ แล้วใช้ AES เข้ารหัสไฟล์รูปและไฟล์ text ในโฟลเดอร์ (ระบุชื่อไฟล์เป้าหมายที่แน่นอนในโปรแกรม ***เพื่อป้องกันความผิดพลาดไปเข้ารหัสไฟล์ที่ไม่ต้องการนอกโฟลเดอร์***)
➢ สุดท้ายโปรแกรมจะสร้างไฟล์ text ใหม่เพื่อเก็บกุญแจ AES และ IV พร้อมเข้ารหัสไฟล์ text นี้ด้วย Public key ที่สร้างไว้ตอนต้น (ไฟล์ชื่อ LocalKey)
• 2. เมื่อเลือกโหมด Decryption จะให้ใส่ Private Key ที่สร้างไว้ตอนต้น
➢ โปรแกรมจะใช้ Private Key ถอดรหัสไฟล์ LocalKey และแสดงผลกุญแจ AES ที่ถอดรหัสออกทางหน้าจอ 1 ครั้ง
➢ ถอดรหัสไฟล์รูปและไฟล์ text ในโฟลเดอร์ กลับมาเหมือนเดิมได้
➢ ทดสอบด้วยการเปิดไฟล์ทั้งสองว่าดูได้ไหม
สิ่งที่ต้องส่ง (กลุ่มละ 2 คน)
1. Source code
2. Video แสดงการเข้ารหัสและถอดรหัสพร้อมอธิบายหลักการของโปรแกรม (ไม่เกิน 5 นาที)
3. ไฟล์ text เก็บค่า Private key และ Public key (ไม่ต้องเข้ารหัส)
4. ไฟล์ต้นฉบับทั้ง 2 ไฟล์ (ไฟล์รูปและไฟล์ text)
